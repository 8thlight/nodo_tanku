## Welcome to Nōdo Tanku (ノード タンク)

### Development Requirements
* [node](https://github.com/ry/node) 0.4.0 or higher (tested on 0.4.0)
* [npm](https://github.com/isaacs/npm)

### Starting local server
<pre>$ node server.js</pre>
